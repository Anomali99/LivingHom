# LivingHome

> python + flask + sqlite

## Technology stack

- Python
- Flask

## Init

### 1. Create venv

```bash
python -m venv venv
```

### 2. Activate venv

for windows :

```bash
venv\Scripts\activate
```

for linux :

```bash
venv\bin\activate
```

### 6. Install requirements

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```
